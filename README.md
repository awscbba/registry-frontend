# Registry Frontend

Astro-based frontend application for the People Registry system.

## Quick Start

```bash
npm install
npm run dev
```

## Documentation

All documentation is maintained in the `registry-documentation` repository.

## Repository Structure

- `src/` - Source code
- `public/` - Static assets
- `dist/` - Built application

For detailed documentation, setup instructions, and architecture information, see the `registry-documentation` repository.
